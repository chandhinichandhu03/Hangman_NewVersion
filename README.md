# 🎮 Hangman Quest

**Hangman Quest** is a premium, web-based word guessing game with a secure authentication system, real-time progression, and a smooth gaming UI.

## 🚀 Features

-   **🔐 Secure Authentication**: Signup and Login with password hashing (Bcrypt) and session handling (JWT).
-   **🏠 Dashboard**: Personalized welcome, high scores, and game history.
-   **🎯 Game Customization**: Choose from 4 categories (Programming, Movies, Animals, Countries) and 3 difficulty levels (Easy, Medium, Hard).
-   **🎨 Premium UI**: Glassmorphism design, smooth animations, and responsive layout.
-   **✍️ Interactive Gameplay**: Animated hangman SVG, on-screen keyboard, and sound effects.
-   **🏆 Scoring & Leaderboard**: Earn points for correct guesses and track your position against other players.

## 🛠️ Tech Stack

-   **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
-   **Backend**: Node.js, Express.js
-   **Database**: SQLite (SQL)
-   **Auth**: JWT, BcryptJS

## 🏃 Getting Started

### Single Command to Run (macOS & Windows)

Run this single command in your terminal (macOS/Linux) or Command Prompt/PowerShell (Windows) to install all dependencies and start the local server:

```bash
npm install && npm start
```

Once started, open [http://localhost:3000](http://localhost:3000) in your browser.

## 🎮 How to Play

1.  Sign up or Login to your account.
2.  In the dashboard, select your preferred category and difficulty.
3.  Click **START QUEST**.
4.  Guess the word by clicking letters or using your physical keyboard (interactive buttons).
5.  If you make 6 wrong guesses, the hangman is complete and the game is over!
6.  Win to earn bonus points and climb the leaderboard.
